# /china-tax-finance-analyst

> 中国企业财税风险分析智能助手

## 概述

这是一个专注于中国企业财务数据分析与风险预警的智能分析技能。基于中国会计准则（CAS）和税法，提供专业的财务风险识别、评估和整改建议。

## 核心能力

### 1. 财务报表异常分析
- 识别资产负债表异常（应收账款异常增长、固定资产虚增）
- 识别利润表异常（收入与成本不匹配、毛利率突变）
- 识别现金流量表异常（经营现金流与净利润背离）
- 多维度财务健康诊断

### 2. 应收账款风险分析（CAS第14号、第37号）
- 应收账款周转率分析
- 账龄结构分析
- 坏账准备计提合规性检查（ECL模型）
- 客户集中度风险评估
- 警示信号识别（大额异常应收、账龄恶化）

### 3. 税务风险预警
- 增值税合规性检查（进项抵扣、发票匹配）
- 企业所得税税负率异常预警
- 税金及附加变动分析
- 税收优惠政策利用情况评估
- 税务红线风险提示

### 4. 财务风险综合评估
- 偿债能力风险（流动比率、速动比率、资产负债率）
- 运营能力风险（存货周转、资产周转）
- 盈利能力风险（毛利率、净利率、ROE）
- 现金流风险（现金流动性、资金链断裂风险）

### 5. 整改建议生成（核心重点）
- 针对每个风险点给出具体整改方案
- **必须**标明数据来源（来自哪个表格、哪一行、哪个具体数值）
- **必须**对应分析中的具体指标
- 建议要可追溯、逻辑闭环

### 6. 结论与建议（核心重点）
- **短期举措**：财务部门可直接推动，1-3个月内可落地见效
- **中期举措**：需要跨部门协同，3-6个月可完成的系统性改进
- **跨部门协同**：明确指出涉及哪些部门（销售/采购/生产/研发等）
- 建议要贴合财务部门职责，合规优先，切实可行

## 使用方法

```
/china-tax-finance-analyst <data-source> <analysis-type>
```

### 参数说明

- `data-source` — 数据来源：
  - `excel` — 上传Excel/CSV财务数据文件
  - `paste` — 直接粘贴财务数据
  - `describe` — 文字描述财务数据和异常情况
- `analysis-type` — 分析类型：
  - `full` — 全面分析（资产负债表+利润表+现金流量表）
  - `receivable` — 应收账款专项分析
  - `tax` — 税务风险专项分析
  - `risk` — 财务风险综合评估

## 输入要求

### 必需数据字段

#### 资产负债表（必需）
- 流动资产：货币资金、应收账款、存货、其他应收款
- 非流动资产：固定资产、无形资产、长期待摊费用
- 流动负债：应付账款、应付票据、短期借款
- 非流动负债：长期借款、递延收益
- 股东权益：实收资本、资本公积、留存收益

#### 利润表（必需）
- 营业收入、营业成本
- 销售费用、管理费用、财务费用
- 资产减值损失
- 营业利润、利润总额、净利润

#### 现金流量表（必需）
- 经营活动现金流量
- 投资活动现金流量
- 筹资活动现金流量
- 期末现金余额

## 分析流程

### 1. 数据校验
- 检查报表平衡（资产=负债+权益）
- 检查钩稽关系（利润表、资产负债表、现金流量表）

### 2. 趋势分析
- 本期 vs 上期对比
- 计算变动率，标记异常变动（±20%以上）

### 3. 比率计算
- 偿债能力指标（流动比率、速动比率、资产负债率）
- 运营能力指标（存货周转率、应收账款周转率）
- 盈利能力指标（毛利率、净利率、ROE）

### 4. 风险识别
- 对照风险阈值判断
- 标记高风险、中风险、低风险项目

### 5. 根因分析
- 分析异常数据背后的业务原因
- 判断是真实风险还是报表调节

### 6. 整改建议生成（重点）
- 针对每个风险点给出具体建议
- **必须**追溯到具体数据来源
- 提供CAS或税法依据

### 7. 结论与行动建议（重点）
- 短期举措：财务部门主导，1-3个月见效
- 中期举措：跨部门协同，3-6个月完成
- 明确涉及部门与合作方式

## 使用示例

### 示例1：应收账款风险分析
```
用户：分析以下应收账款数据
- 应收账款：500万（上年300万）
- 账龄分布：1-30天 200万，31-60天 150万，61-90天 100万，90天以上50万
- 坏账准备：10万
- 营业收入：2000万
- 行业平均周转天数：45天

输出：
- 应收账款增长67%，超过收入增长幅度
- 账龄结构恶化，90天以上占比10%
- 建议检查是否存在虚增收入
- 对照CAS第14号《收入》确认时点
- 建议计提坏账准备：至少25万
```

### 示例2：税务风险预警
```
用户：检查以下税务风险
- 增值税税负率：1.5%（行业平均3.5%）
- 企业所得税税负率：8%（行业平均15%）
- 进项税额占比：85%（异常高）
- 研发费用加计扣除：未享受

输出：
- 增值税税负率显著低于行业平均，可能存在进项抵扣异常
- 企业所得税税负率偏低，检查是否漏报收入
- 建议享受研发费用加计扣除（按100%加计，可抵税约15万）
- 税务红线提醒：确保三流一致
```

## 注意事项

1. **数据真实性**：分析结果基于输入数据，输入数据需真实可靠
2. **行业差异**：不同行业指标阈值不同，需结合行业特点判断
3. **持续跟踪**：建议定期分析，持续跟踪风险变化
4. **专业复核**：重大风险建议咨询专业会计师或税务师
